Part 2 — Detailed User Flow
Screen-by-screen breakdown. Every screen has: what's on it, what the user does, what happens next, and any technical notes for the developer.


Screen 1 — Splash Screen

Screen 2 — Onboarding (3 slides, swipeable)



Screen 3 — Permissions

Screen 4 — Setup (Skippable, can return any time)


Screen 5 — Home Screen
The main screen. Designed for single-thumb use. No menus. No search. Just the 6 big buttons.



Screen 6 — Triage Screen
Three yes/no questions. Fast. Large buttons. Used to route user to right level of care.


Screen 7 — Emergency Screen (Medical Mode)


Screen 8 — Evacuation Guide
Shown immediately when user taps Car Fire. Before showing any services — first tell them what to do in the next 90 seconds.


Screen 9 — Emergency Screen (Fire Mode)


Screen 10 — Breakdown Screen
Used for both Breakdown and Fuel flows. When accessed from Fuel button, the 'Petrol Pumps' tab is pre-selected.



Screen 11 — Emergency Screen (Flood Mode)


Screen 12 — Silent SOS Screen
This screen must be designed carefully. It should appear completely inactive to someone looking at the phone from outside.



Screen 13 — Bystander Entry
Accessed via the 'I'm a bystander' strip on Home Screen, OR from Emergency Screen of any active incident.


Screen 14 — Role Assignment Screen (Incident Merge)

Screen 15 — Bystander Guidance Screen


Screen 16 — Medical QR ID Screen
Accessible from Setup Screen or Profile tab. QR is generated entirely offline.



Screen 17 — Safe Arrival Setup
Accessed from Setup Screen or Profile tab. Designed for use before a long highway drive.



Screen 18 — KM Marker Input (available on Emergency + Breakdown screens)
Shown as a small link on any screen that displays location: 'Don't know your GPS address? Enter km marker'. Critical for highway emergencies.



Screen 19 — Incident Map Screen
A shareable web URL (roadsos.app/incident/[id]) that opens as a screen inside the app. Family members can open it in browser.
